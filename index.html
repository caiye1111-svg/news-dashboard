瑞士议会正式解除核电站建设禁令，欧洲能源格局迎来转折点

🌍 国际 · ⚡ 能源 · 🇨🇭 瑞士
瑞士议会以压倒性票数通过法案，正式废除自 2017 年公投以来的新建核电站禁令。此举标志着欧洲在经历能源危机后对核电态度出现重大转向。瑞典、比利时、荷兰等国近期也相继重启或延长核电站计划，而德国仍坚持 2023 年全面弃核路线——欧洲内部能源路线的分裂正在加深。
🔥 HN 662 points · 515 评论 📎 bluewin.ch · HN #7
瑞士「核电大转弯」背后是欧洲能源自主焦虑的集中爆发。2022年天然气危机以来，欧盟国家对核电的态度已从「意识形态站队」转向「务实生存计算」。全球铀矿股和核能基础设施供应链将受益——但德国坚持弃核意味着欧洲电网互联成本将大幅上升，最终买单的是工业用户。

🤖 Anthropic Mythos 争议升级：白宫干预模型发布，安全 vs 创新之争白热化🌐 HN+Verge

🤖 AI · ⚖️ 政策 · 🇺🇸 美国
Anthropic 最新 Claude Mythos 5（代号 Fable）被白宫以「过于危险」为由禁止公开发布，仅限 CISA 等安全机构访问。The Verge 深度调查揭示，此次干预背后是 Amazon 安全研究团队向白宫提交的风险评估报告。韩国电信巨头 SK Telecom 也被卷入争议——作为 Anthropic 主要投资者之一，其在 Mythos 训练数据中的角色引发质疑。HN 社区激烈讨论：到底谁来定义 AI「太危险」？
"If the US government can unilaterally block a model release, every other country will accelerate their own AI programs to avoid dependence. This is how you lose AI leadership."
— HN 用户 scotty79，55 条讨论中排名第一
风险信号：Mythos 事件标志着美国 AI 监管从「自愿承诺」进入「行政干预」新阶段。对 Anthropic 短期利空（收入受损），长期利好（安全标杆地位）。但非美国 AI 公司（DeepSeek、Mistral）将从「监管套利」中获益——这正是 Robert Hart 在 The Verge 评论中指出的核心悖论。

🛡️ 安全研究员发现超 10,000 个 GitHub 仓库传播木马病毒

🤖 安全 · 🔐 开源 · ⚠️ 供应链
安全研究者 theorchid 在 HN 披露：通过自动化扫描发现超过 10,000 个 GitHub 公开仓库正在分发 Trojan 恶意软件，涉及虚假 PoC 利用代码、被植入后门的流行库克隆版本等。该发现获得 635 points 和 144 条评论，引发对开源供应链安全的广泛担忧。研究者已向 GitHub 安全团队报告。
"This is terrifying. How many of these have been automatically pulled into CI/CD pipelines without anyone noticing? The open source supply chain is a disaster waiting to happen."
— HN 用户 pjmlp
开源生态的系统性风险正在暴露：AI 编码工具（Copilot、Cursor）大量依赖 GitHub 代码训练和引用，若恶意仓库未被及时清理，污染可能通过 AI 辅助编码扩散到企业代码库。GitHub 的内容审核能力正在被 AI 时代的供给速度远远甩开。

💰 百胜中国 12 亿美元收购必胜客中国大陆品牌所有权

💰 商业 · 🍕 餐饮 · 🇨🇳 中国
百胜中国宣布以 12 亿美元从百胜全球手中收购必胜客在中国大陆的完整品牌所有权和知识产权。这意味着必胜客在中国将完全独立运营，不再向百胜全球支付品牌授权费。此举被知乎热议，分析认为此举既降低长期成本（每年节省数千万美元授权费），也赋予百胜中国更大的本地化创新空间。
"必胜客在中国早就'中国化'了——小龙虾披萨、酸菜鱼披萨这些都不是美国总部能理解的。买断品牌是迟早的事。"
— 知乎用户 @商业望远镜，1200+ 赞同
利好百胜中国：12亿美元一次性支出换来永久品牌自主权，考虑到必胜客中国年营收超20亿美元，ROI极高。但这也意味着百胜全球正在战略收缩——将区域品牌资产变现回笼资金，背后可能是对中美脱钩长期化的预判。

🚀 SpaceX 正式以 60 亿美元收购 Cursor，AI 编码工具最大并购案落定

🚀 Musk · 🤖 AI · 💰 并购
The Verge 确认 SpaceX 已完成对 AI 编程工具 Cursor 的收购，交易金额 60 亿美元。这是迄今为止 AI 编码工具领域最大的一笔并购。Cursor 拥有数百万开发者用户，其 AI-first IDE 理念一直被视为 GitHub Copilot 最强竞品。交易完成后 Cursor 团队将并入 SpaceX 的 Starlink 和 Starship 软件开发体系。
"Musk buying an IDE is the most Musk thing ever. Next he'll buy Stack Overflow and make every answer require an X Premium subscription."
— The Verge 评论区热评
短期中性：SpaceX 是非上市公司，Cursor 将不再独立服务外部开发者，用户可能流向 Copilot/Windsurf。长期看，Musk 旗下 xAI + Cursor + X 数据形成闭环——这是「AI+开发工具+社交数据」的垂直整合实验，若成功将重塑开发者工具市场格局。
